# 🚀 AICodeAgent

**AICodeAgent** is a premium, enterprise-grade Local AI Fullstack Engineer and System Architect. It enables end-to-end application development, code generation, and intelligent debugging—all running locally on your hardware for 100% privacy and $0 cost.

## ✨ Key Features

- **🏗️ End-to-End Development**: Generate production-ready full-stack applications from a single natural language prompt.
- **💻 Smart Code Generation**: Queries-based code updates, specific feature implementations, and logic generation.
- **🪲 AI-Driven Debugger**: Deep analysis of existing files or pasted code blocks to identify, explain, and fix bugs instantly.
- **🦙 Ollama Integration**: Powered by your local Ollama instance. Supports `llama3`, `codellama`, `mistral`, and more.
- **💎 Premium Aesthetics**: A sleek "Obsidian Dark" UI featuring glassmorphism, fluid animations, and high-fidelity code previews.
- **🔒 Privacy First**: Your proprietary code never leaves the local machine.

## 🚦 Quick Start

### 1. Prerequisites
Ensure you have **Ollama** installed and running on your local machine.
- [Download Ollama](https://ollama.com/download)
- Pull a high-performance model:
```bash
ollama pull llama3
```

### 2. Installation
Clone the repository and install dependencies:
```bash
npm install
```

### 3. Running Locally
Start the development server:
```bash
npm run dev
```
Navigate to [http://localhost:3000](http://localhost:3000) (or the port specified in terminal).

## 📖 Usage Modes

1. **New Project**: Best for "blue-sky" development. Describe what you want to build and let the AI architect the entire folder structure.
2. **Code Query**: Provide a specific query about an existing component or ask the AI to write a specific function.
3. **AI Debugger**: Paste your problematic code or upload your local files to receive optimized, bug-free versions.
